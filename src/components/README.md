# Components

## `ScheduleTable`

### props
#### `mainHeaders`
- Type: `[ScheduleHeader]`
- Details:

  Headers used in the main axis, also used to access grouped items.

#### `crossHeaders`
- Type: `[ScheduleItem]`
- Details

  Headers used in the cross axis.

#### `items`
- Type: `[ScheduleItem]`
- Details:

  Items included in the table.
